# REPLICATE-CER

**Coordinated multi-site, equity-enabling replications for Computing Education Research (CER).**

Computing education needs credible, generalizable evidence. Replications—done rigorously and across diverse contexts—let us build confidence in results and lay foundations for computing-specific educational theory.

**What we do**
- Organize **cohorts** to design and run coordinated replications on key CER topics
- Provide a **replication package** (research questions, data & analysis plan, IRB template, timeline)
- Support **public preregistration**, responsible **data sharing**, and **meta-analysis**
- Train and mentor researchers to conduct rigorous empirical CER

**Current focus (Cohort 1):** Assessing individual **team contributions** in software engineering/capstone contexts.
